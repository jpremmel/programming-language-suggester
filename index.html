JavaScript!

Anyone interested in front-end design work should learn HTML and CSS - and after learning those, JavaScript is the logical next step for taking your webpages from being static to interactive. Unlike HTML and CSS, JavaScript is a full programming language. It is used to fuel a website's behavior. JavaScript has become increasingly popular, and it now lives outside web browsers as well. Learning JavaScript will put you in a good place as it becomes a more general-purpose language.
